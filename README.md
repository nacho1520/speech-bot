# Speech Bot

Speech Bot page made with HTML, SASS and Vanilla JavaScript. 

User can enter a text and listen to the bot. Also, user can change the bot voice language and rate. 

## Technologies:

* HTML
* SASS
* JavaScript
* SpeechSynthesisUtterance
